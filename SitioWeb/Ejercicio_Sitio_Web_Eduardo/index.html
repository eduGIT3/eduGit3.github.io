Esta web ha sido creada para la asignatura de lenguajes de marcas con el motivo de realizar la tarea 6.1 Aquí de este espacio, se analizarán los conceptos clave de los SGE, sus principales tipologías y las características que los definen. Además, se documentará de forma práctica la instalación, configuración y administración de una solución de software libre, evaluando aspectos críticos como la seguridad de la información, la gestión de roles de usuario, y la capacidad para elaborar y exportar informes estratégicos.
